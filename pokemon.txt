contoh pagination di sequelize
static async getProduct(req, res, next) {
  const page = req.query.page ? Number(req.query.page) : 1; // Get the requested page number
  const limit = 50; // Set the number of items to show per page

  try {
    const data = await Product.findAll({
      include: {
        nested: true,
        all: true,
      },
      offset: (page - 1) * limit, // Calculate the offset based on the requested page number
      limit: limit, // Set the limit to the number of items per page
    });

    res.status(200).json({ data });
  } catch (error) {
    console.log(error);
    res.status(500).json({ message: "Internal Server Error" });
  }
}
In this example, the page variable is used to get the requested page number from the query parameters. The limit variable is set to the number of items to show per page (in this case, 50). The offset option is calculated based on the requested page number, and the limit option is set to the limit variable. This will retrieve only 50 items at a time, starting from the offset specified.

Note that you may also want to include additional logic to handle cases where the requested page number is out of range or when the page query parameter is not provided.
